Introduction

What is EverRun?

EverRun is a cloud-based website deployment platform designed for developers and businesses - similar to Vercel, Netlify, and Render.

✓ Deploy static sites from ZIP files
✓ Deploy from GitHub repositories
✓ Smart framework detection (Node.js, Django, Laravel)
✓ Docker containerization
✓ Real-time deployment logs
✓ GitHub OAuth authentication
✓ Stripe payment ready

Coding Standards

Project Structure

everrun/                         # Root Project Directory
├── backend/                      # Django Project
│   ├── config/                   # Project Settings
│   │   ├── settings.py           # Django settings
│   │   ├── urls.py               # URL routing
│   │   ├── wsgi.py               # WSGI entry
│   │   └── asgi.py               # ASGI entry
│   ├── deployments/              # Main Deployment App
│   │   ├── models.py             # Site model
│   │   ├── views.py              # All views
│   │   ├── urls.py               # URL routing
│   │   ├── forms.py              # Upload forms
│   │   ├── deployer.py           # SmartDeployer class
│   │   ├── adapters.py           # AllAuth adapter
│   │   └── admin.py              # Django admin
│   ├── templates/                 # HTML templates
│   └── manage.py                 # Django management
├── sites-data/                   # Deployed sites data
├── releases/                     # Release versions
├── scripts/                      # Deployment scripts
├── requirements.txt              # Python dependencies
└── readme.md                    # Documentation

Coding Best Practices

  • Follow PEP 8 Python style guide
  • Use meaningful variable and function names
  • Use template inheritance
    • logic minimal in views
  • Use decorators for authentication
  • Keep business
  • Never commit secrets to version control
